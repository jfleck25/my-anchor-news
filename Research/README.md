# Research

This directory contains competitive analysis, API comparisons, and market research.

## Structure

- **voice-apis/** - Text-to-speech API comparisons (Google TTS, AWS Polly, ElevenLabs, etc.)
- **competitors/** - Analysis of similar products (Briefing, Morning Brew, etc.)
- **market/** - Industry trends, user behavior research

## Guidelines

- Include pros/cons for each option
- Note pricing, limits, and technical constraints
- Keep research current with dates
